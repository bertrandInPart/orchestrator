# Conventions memory

Accumulated, cross-feature naming and scoping conventions. Agents append one line (or short
section) here when they make a reusable decision worth remembering for future features — this is
the PROSE "Memory" primitive: durable context that saves the next agent from re-deriving the same
answer from scratch.

Append-only in spirit: prefer adding a new dated entry over editing an old one, unless a
convention is being explicitly superseded (state that explicitly when it happens).

Format:

```
## <YYYY-MM-DD> — <short title>
<one or two sentences: what the convention is, and which feature/slug introduced it>
```
